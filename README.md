# apollo_vision
This stack contains vision related packages such as:
- darknet_ros (https://github.com/leggedrobotics/darknet_ros)
- img_converter
- mediapipe_track

## Installation
```git clone --recurse-submodules https://github.com/UtBot-UTFPR/apollo_vision.git```
